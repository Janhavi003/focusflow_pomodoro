# FocusFlow ⏱️

FocusFlow is a modern **Pomodoro productivity web app** designed to help users stay focused, track work sessions, and improve productivity.

It combines the Pomodoro technique with task management, productivity analytics, focus sounds, and a clean modern UI.

---

## 🚀 Features

### Pomodoro Timer
- 25 minute focus sessions
- 5 minute break sessions
- Automatic session switching
- Alarm sound for focus and break completion
- Circular animated progress timer
- Floating draggable timer widget

### Task Management
- Add tasks
- Delete tasks
- Select active task
- Track completed pomodoros per task

### Daily Goals
- Set daily Pomodoro goal
- Track completed sessions
- Visual progress bar

### Productivity Analytics
- Daily focus session chart
- Weekly productivity trend
- Built using Recharts

### Session History
- Stores completed focus sessions
- Displays date, task, and duration
- Stored in LocalStorage

### Focus Sounds
- Rain ambience
- Forest ambience
- Cafe ambience
- White noise
- Volume control

### UI / UX
- Modern dashboard layout
- Sidebar navigation
- Responsive design
- Dark / Light mode toggle
- Draggable floating timer

### Landing Page
- Modern SaaS-style landing page
- Hero section
- Features section
- How it works section
- Call-to-action
